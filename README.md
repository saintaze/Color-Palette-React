# Color-Palette-React
An app for creating colors.

### Features
+ Responsive Design
+ Change Color Scale
+ Create new Palettes
+ Route Transitions
+ Local Storage to persist data
+ Draggable Color boxes

##### Built With
+ HTML5
+ CSS3
+ React

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `npm start`
